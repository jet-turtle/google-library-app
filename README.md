Google Library

An Android app integrating Google Books API to search global library databases with seamless redirection to external purchase pages.

Tech Stack

    UI: Jetpack Compose (Declarative UI).

    Architecture: MVVM (Model-View-ViewModel).

    DI: Manual Dependency Injection (for lightweight performance and transparency).

    Network: Retrofit.
